# 문제 풀이

### 2678. Number of Senior Citizens ```EASY```
- 승객 정보(전화번호 + 성별 + 나이 + 좌석) 목록이 주어졌을 때 60세가 넘는 승객의 수를 구하는 문제
- 승객 정보에서 11, 12번째 문자를 찾아서 나이인 숫자로 변환한 뒤, 60이 넘는 경우를 찾아 그 수를 반환하여 문제를 해결할 수 있습니다.
- 시간 복잡도: O(n) (n <= 100)
